# 🏗 Scaffold-ETH - BG HACK


```bash
git clone -b bg-hack https://github.com/scaffold-eth/scaffold-eth bg-hack
```

> install and start your 👷‍ Hardhat chain:

```bash
cd bg-hack
yarn install
yarn chain
```

> in a second terminal window, start your 📱 frontend:

```bash
cd scaffold-eth
yarn start
```
